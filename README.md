# 🏪 Student BizDir - Campus Business Directory

A web platform that connects students with student-run businesses on campus. Find skincare products, food services, tech repairs, and more - all run by your fellow students!

## 📋 The Problem

Students on campus run small businesses but struggle to reach their target audience (other students). Currently, they rely on:
- WhatsApp status updates that get lost
- Posters on notice boards that get torn down
- Word of mouth with limited reach

**Student BizDir** solves this by creating a centralized, searchable directory of all student businesses.

## ✨ Features

- 🔍 **Search businesses** by name, category, or description
- 🏷️ **Filter by category** (Food, Tech, Beauty, Services, etc.)
- 📱 **Mobile-responsive design** - works on phones and laptops
- 📞 **One-click contact** via WhatsApp integration
- 📝 **Business registration** form for new vendors
- 🤖 **AI-powered search** using Claude API (find businesses naturally)

## 🛠️ Tech Stack

- **Backend:** Python + Flask (lightweight web framework)
- **Database:** SQLite (simple, file-based database)
- **Frontend:** HTML, CSS, Bootstrap 5
- **AI Integration:** Anthropic Claude API
- **Version Control:** Git & GitHub
